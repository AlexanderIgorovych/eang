ARIA Roles (Accessible Rich Internet Applications), defines a way to make Web content and Web applications more accessible. We used it, along with its origonal intention, as a basis for commonly reused styles. The goal is to allow the developer to use ARIA roles instead of potentially redundant classes.

role="button"

The Button role allows you to achieve consistency with anchor tags by making them look like buttons. This borrows from the button component and inherits all the additional attributes and features.
