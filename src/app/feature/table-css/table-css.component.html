Table

Add the ea-table class to tab <table>. The class will add padding, border and emphasized table header. You can changed almost all table settings (see css variables at the bottom of the article). For table header: color, text-alignement, border color etc. For table row: color, text-alignement, border color etc. There are 4 tables with different default settings.
