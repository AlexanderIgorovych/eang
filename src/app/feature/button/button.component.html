Buttons

Eang uses a combination of em, custom properties, and the calc() function to make implementing proportional sized buttons more simple. It capitalizes on the inherit features of em that allow elements of the button to scale relative to a set font-size defined within the component.

<button>

These variables can all be adjusted to customize aspects of your button that you want to scale with. We've added a few attributes to allow you to easily scale up and down to preset measurements.

Adding Icons

Eang makes it easy to add icons into your buttons with a preset lay out that also scales up proportionally based off font-size. the [icon] attribute can be added to a button to take care of some basic icon work.

<button>

Setting the icon in the source code of our application is very simple. Using one of the pre-defined custom properties that represent an icon from icons.css you can quickly set the button to have an icon.
