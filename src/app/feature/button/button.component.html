Buttons

Buttons allow users to take actions, and make choices, with a single tap. Eang uses a combination of em, custom properties, and the calc() function to make implementing proportional sized buttons more simple. It capitalizes on the inherit features of em that allow elements of the button to scale relative to a set font-size defined within the component.

Icon Buttons

Eang makes it easy to add icons into your buttons with a preset lay out that also scales up proportionally based off font-size. the [icon] attribute can be added to a button to take care of some basic icon work.
