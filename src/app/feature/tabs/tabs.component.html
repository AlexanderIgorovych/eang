Tabs

Tabs are a set of layered sections of content, known as tabpanels, that display one tabpanel of content at a time. Each tab panel has an associated tab element, that when activated, displays the tabpanel. Unlike other angular tab implementations, eang does not require tabs and tabpanels to be coupled. <ea-tablist> and the associated <ea-tabgroup> can be placed anywhere on the page.

Components

<ea-tablist>

Contains the tabs that can be activated to display different content in the tabgroup.

<ea-tabgroup>

Collection of tabpanels.

<ea-tabpanel>

Tabpanel is defined within the tabgroup. The name attribute is mandatory.
