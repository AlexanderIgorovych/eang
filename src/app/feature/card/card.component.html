Card Sizing

Cards can be dynamically sized using the eang utility attributes of sm, md, lg, and xl. This will proportionally scale interior and exterior margins, padding, and font-sizes of any ea-card.

.ea-card-content

The .ea-card-content is meant to hold the content of the card. It is designed to be a column and wraps content that overflows horizontally automatically to accomodate any size content and to create easy responsive designs.

.ea-card-footer

The card footer is designed to hold action buttons and a message. The layout is designed to hold 1-3 elements without any adjustment.

.ea-card[unset]

The [unset] attribute is used to void all of the visual properties of a card, but maintain the layout properties. This allows the ea-card to be used for layout purposes.
