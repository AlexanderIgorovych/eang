Table

To improve the view of your table just add .ea-table class to <table> tag and it will add padding, border and emphasized table header. You can changed almost all table settings (see css variables at the bottom of the article). For table header: color, text-alignement, border color etc. For table row: color, text-alignement, border color etc. Four tables with different default settings are shown below.

Components

Striped and Hover Table

Just use class ea-table in <table> tag. You can mix both striped and hovered. Just add both attributes: striped and table-hover in <tbody>.
