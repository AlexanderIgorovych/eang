Reactive

Status changes in enterprise applications need to be updated in real-time. Eang components are designed to react to events.

These UI components interact with each other through composable event streams. This allows developers to implement more rapidly and reach a higher level of complexity in their applications.

The following example shows the ea-panel component reacting to two event stream observables, panel state and orientation.

<ea-panel>
