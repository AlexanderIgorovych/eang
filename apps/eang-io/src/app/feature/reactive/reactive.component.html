Reactive

Status changes in enterprise applications need to be updated in real-time. Eang components deliver on that promise by bringing basing UI components of event streams.

UI components and their interactions that based off of componsible event streams allow developers to implement more rapidly and reach a new level more quicly of application interaction.

Each component in eang can be directly hooked up to event streams. They observe changes and react to them. The main benefit of reactive components is that you can compose them very easily through streams. This ways you can combine mutiple ways on how to interact with your component.
